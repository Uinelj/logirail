Logotype
========

La police utilisée est [Roboto](https://www.google.com/fonts/specimen/Roboto), sous licence Apache2 ( voir le fichier `LICENCE.md` ).

Les couleurs utilisées sont :

Gris : `#393939`

Le 'A' : `#fe0009`

Le cercle : `#0e3e80`

Feu de signalisation : 

* Rouge : `#fe666b`
* Orange : `#ffb085`
* Vert : `#77b868`
